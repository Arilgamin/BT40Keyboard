# Aril40Keyboard

## 40% Mechanical Keyboard BLE

Custom mechanical keyboard 40% menggunakan ESP32-C3 SuperMini.

## Spesifikasi

- MCU : ESP32-C3 SuperMini
- Interface : Bluetooth BLE
- Matrix : 4 Row × 12 Column
- Switch : MX Style
- Diode : 1N4148
- Layout : 40%
- Firmware : Custom Arduino

## Fitur

- Bluetooth Keyboard
- NKRO (Rencana)
- Layer Fn
- Angka
- Simbol
- F1–F12
- Arrow Key
- Auto Reconnect
- Low Power Mode

## Matrix

```
4 Rows
12 Columns
48 Keys
```

## Hardware

ESP32-C3 SuperMini

```
R0 = GPIO0
R1 = GPIO1
R2 = GPIO2
R3 = GPIO3

C0  = GPIO4
C1  = GPIO5
C2  = GPIO6
C3  = GPIO7
C4  = GPIO8
C5  = GPIO9
C6  = GPIO10
C7  = GPIO11
C8  = GPIO12
C9  = GPIO13
C10 = GPIO18
C11 = GPIO19
```
